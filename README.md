## Prerequisites

1. Sign up for a [free Twilio account](https://www.twilio.com/try-twilio)
2. [ngrok](https://ngrok.com/) or [VSCode port forwarding](https://code.visualstudio.com/docs/debugtest/port-forwarding) to expose local ports to the internet
