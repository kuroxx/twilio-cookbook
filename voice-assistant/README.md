# Twilio Cookbook 🍳

A masterclass demo project for building a culinary AI assistant using Twilio.

## Setup

1.  **Clone the repo**
2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Environment Variables**:
    Create a `.env` file with:
    ```
    TWILIO_ACCOUNT_SID=your_sid
    TWILIO_AUTH_TOKEN=your_token
    ```
4.  **Run the Notebook**:
    ```bash
    jupyter notebook twilio_cookbook.ipynb
    ```
